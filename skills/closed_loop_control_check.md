# Skill: Closed-Loop Control Check

Use after designing loop compensation.

Inputs:
- Plant parameters, PID/Type-3 gains, Bode data, transient response.

Steps:
- Check crossover, phase margin, gain margin, overshoot, undershoot, settling, and oscillation count.
- Confirm results are simulation-backed before signoff.

Human approval:
- Required for protection threshold or gate timing changes.
