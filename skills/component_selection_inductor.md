# Skill: Inductor Selection

Use when selecting off-the-shelf inductors or planning custom magnetics.

Inputs:
- Inductance target, DC current, ripple current, switching frequency, thermal limit.

Steps:
- Check saturation current, RMS current, DCR, core loss, height, footprint, and thermal rise.
- Prefer datasheet loss curves; otherwise mark core loss low-confidence.

Human approval:
- Required before using placeholder core loss for release decisions.
