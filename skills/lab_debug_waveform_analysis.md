# Skill: Lab Debug Waveform Analysis

Use when analyzing oscilloscope captures, current logs, thermal images, or telemetry.

Inputs:
- Waveform files, operating point, schematic/layout context, protection state.

Steps:
- Identify overshoot, ringing, saturation, current-limit behavior, instability, thermal stress, and measurement setup risks.
- Recommend safe next measurements only.

Human approval:
- Required for high-voltage or high-current physical tests.
